職稱：軟體工程師

職責：

  • Participate in technical design, analysis, implementation and testing of software product using a variety of technologies including C#, ASP.NET MVC, Web Services, Microsoft SQL Server, JavaScript, TypeScript, React, GraphQL, Microsoft .NET Core, and Node.js.
  • 負責不同低代碼業務流程管理（BPM）軟體開發解決方案的產品範疇、需求收集、架構設計、開發和交付。
  • 使用各種資料庫管理軟體，如 Microsoft SQL Server、Oracle、MySQL、Amazon Dynamo DB，設計、實施並維護產品資料庫。
  • 使用 Gitlab、TFS、Azure DevOps、Docker 和 Kubernetes 來設計、實施及管理產品開發和部署流程（DEVOPS）。
  • 使用 AgilePoint NX 低代碼 BPM 平台，運用 OAuth2、OpenID Connect 和 SAML 協議，設計並實施用於用戶認證與授權的單一登入（SSO）層。
  • 為國內外合作夥伴及新入內部開發團隊成員提供 AgilePoint NX 低代碼 BPM 平台的培訓。在網絡研討會中介紹 AgilePoint 產品。
  • 外部系統與 AgilePoint NX 低代碼 BPM 平臺之間的整合模式之規範與效能需求文件。
  • 與客戶、合作夥伴及內部團隊成員合作，參與軟體產品開發、項目狀態跟進、設計提案及解決技術問題。
  • 與客戶的IT部門合作規劃安裝AgilePoint NX低代碼BPM平台，並為客戶環境中AgilePoint NX低代碼BPM平台及系統的安裝提供最佳實踐建議。

最低要求：

  • Bachelor’s degree or foreign equivalent degree in Electronics and Instrumentation Engineering, Electrical and Computer Engineering, Computer Science and Engineering, Information Science & Engineering, Computer Science, or a related field, and 5 years of post-baccalaureate progressive experience as Software Engineer, Senior Software Engineer, Module Lead, Technical Lead, Product Lead, Senior Development Specialist, Associate Software Architect, Principal Software Engineer, or a related occupation required.

所需的工作經驗必須包括以下的五年經驗：

  • C#、ASP.NET MVC、Web 服務和 GraphQL。
  • Microsoft SQL Server
  • TypeScript、JavaScript、React 和 Node.js
  • Docker，以及
  • 低代碼業務流程管理（BPM）平台的開發。
